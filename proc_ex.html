Thực Hành: Stored Procedure

Trang chủ > Stored Procedure (Bài tập)

Stored Procedure

Tổng hợp danh sách các bài tập rèn luyện kỹ năng Stored Procedure.

🛒 CSDL: Quản Lý Bán Hàng (QUANLYBANHANG)

Bài tập 1

Hiển thị danh các khách hàng đã mua hàng trong ngày chỉ định (ngày là tham số truyền vào). 

CREATE PROCEDURE p_KhachHangTheoNgay @NgayLap date AS BEGIN
SELECT
  KH.*
FROM
  KHACHHANG KH
  INNER JOIN HOADON HD ON KH.MaKH = HD.MaKH
WHERE
  HD.NgayLapHD = @NgayLap END
Tạo thủ tục nhận tham số ngày và thực hiện câu lệnh SELECT với tham số đó.

Bài tập 2

Hiển thị danh sách 5 khách hàng có tổng trị giá các đơn hàng lớn nhất. 

CREATE PROCEDURE p_Top5KhachHang AS BEGIN
SELECT
  TOP 5 KH.MaKH,
  KH.TenKH,
  SUM(C.SoLuong * H.DonGia) AS TongTriGia
FROM
  KHACHHANG KH
  INNER JOIN HOADON HD ON KH.MaKH = HD.MaKH
  INNER JOIN CHITIETHD C ON HD.MaHD = C.MaHD
  INNER JOIN HANGHOA H ON C.MaH = H.MaH
GROUP BY
  KH.MaKH,
  KH.TenKH
ORDER BY
  SUM(C.SoLuong * H.DonGia) DESC END
Sử dụng TOP 5 và ORDER BY DESC để lấy 5 người chi tiêu nhiều nhất.

Bài tập 3

Hiển thị danh sách 10 mặt hàng có số lượng bán lớn nhất. 

CREATE PROCEDURE p_Top10MatHang AS BEGIN
SELECT
  TOP 10 H.MaH,
  H.TenH,
  SUM(C.SoLuong) AS TongSoLuong
FROM
  HANGHOA H
  INNER JOIN CHITIETHD C ON H.MaH = C.MaH
GROUP BY
  H.MaH,
  H.TenH
ORDER BY
  SUM(C.SoLuong) DESC END
Tính tổng số lượng của mỗi mặt hàng, sắp xếp giảm dần và lấy TOP 10.

Bài tập 4

Cập nhật cột Khuyến mãi như sau: Khuyến mãi 5% thành tiền nếu SoLuong >100, 10% thành tiền nếu SoLuong >500. 

UPDATE CHITIETHD
SET
  KhuyenMai = CASE
    WHEN SoLuong > 500 THEN (SoLuong * DonGia) * 0.10
    WHEN SoLuong > 100 THEN (SoLuong * DonGia) * 0.05
    ELSE 0
  END
FROM
  CHITIETHD C
  INNER JOIN HANGHOA H ON C.MaH = H.MaH
Sử dụng lệnh UPDATE kết hợp cấu trúc CASE WHEN.

Bài tập 5

Tính trị giá cho mỗi hoá đơn. 

CREATE PROCEDURE p_TriGiaHoaDon AS BEGIN
SELECT
  C.MaHD,
  SUM(C.SoLuong * H.DonGia) AS TriGia
FROM
  CHITIETHD C
  INNER JOIN HANGHOA H ON C.MaH = H.MaH
GROUP BY
  C.MaHD END
Thủ tục đơn giản sử dụng GROUP BY trên chi tiết hóa đơn.

Bài tập 6

Cập nhật cho cột Loại khách hàng: là VIP nếu tổng thành tiền trong năm lớn hơn hoặc bằng 20 triệu. 

UPDATE KHACHHANG
SET
  LoaiKH = 'VIP'
WHERE
  MaKH IN (
    SELECT
      HD.MaKH
    FROM
      HOADON HD
      INNER JOIN CHITIETHD C ON HD.MaHD = C.MaHD
      INNER JOIN HANGHOA H ON C.MaH = H.MaH
    WHERE
      YEAR(HD.NgayLapHD) = YEAR(GETDATE())
    GROUP BY
      HD.MaKH
    HAVING
      SUM(C.SoLuong * H.DonGia) >= 20000000
  )
Sử dụng UPDATE với điều kiện lọc IN chứa câu truy vấn tính tổng thành tiền.
