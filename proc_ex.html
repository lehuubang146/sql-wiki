Thực Hành: Stored Procedure

Trang chủ > Stored Procedure (Bài tập)

Stored Procedure

Tổng hợp danh sách các bài tập rèn luyện kỹ năng Stored Procedure.

📚 CSDL: Quản Lý Sinh Viên (QLSV_NEW)

Bài tập 1

Thêm Sinh Viên Mới: Viết SP để thêm một sinh viên mới vào bảng SINHVIEN, thực hiện kiểm tra MaLop phải tồn tại trước khi thêm. 

create procedure sp_themsinhvien
    @masv varchar(20),
    @hoten nvarchar(50),
    @malop varchar(10),
    @gioitinh int,
    @ngaysinh date,
    @diachi nvarchar(100)
as
begin
    if exists (select * from dmlop where malop = @malop)
    begin
        insert into sinhvien (masv, hoten, malop, gioitinh, ngaysinh, diachi)
        values (@masv, @hoten, @malop, @gioitinh, @ngaysinh, @diachi)
        print 'them sinh vien thanh cong!'
    end
    else
    begin
        print 'ma lop khong ton tai, khong the them.'
    end
end
Stored Procedure này dùng để thêm mới một sinh viên. Đầu tiên, nó dùng hàm IF EXISTS để kiểm tra xem mã lớp (@malop) người dùng truyền vào có tồn tại trong bảng dmlop hay không nhằm đảm bảo toàn vẹn dữ liệu. Nếu có, nó thực hiện lệnh INSERT INTO để thêm sinh viên và thông báo thành công. Nếu không, nó in ra thông báo lỗi.

Bài tập 2

Nhập/Cập Nhật Điểm: Viết SP cho phép nhập điểm mới cho một sinh viên. Nếu đã có điểm thì cập nhật, nếu chưa có thì thêm mới. 

create procedure sp_nhapcapnhatdiem
    @masv varchar(20),
    @mahp varchar(10),
    @diemhp float
as
begin
    if exists (select * from diemhp where masv = @masv and mahp = @mahp)
    begin
        update diemhp set diemhp = @diemhp where masv = @masv and mahp = @mahp
        print 'da cap nhat diem.'
    end
    else
    begin
        insert into diemhp (masv, mahp, diemhp) values (@masv, @mahp, @diemhp)
        print 'da them diem moi.'
    end
end
SP này quản lý việc nhập hoặc cập nhật điểm. Nó sử dụng IF EXISTS kiểm tra tổ hợp masv và mahp đã có trong bảng diemhp chưa. Nếu đã tồn tại, nó chạy lệnh UPDATE để cập nhật điểm số mới. Nếu chưa tồn tại, nó dùng lệnh INSERT INTO để thêm điểm mới vào.

Bài tập 3

Xuất Bảng Điểm: Viết SP lấy ra danh sách các môn học và điểm số tương ứng của một sinh viên khi biết mã sinh viên. 

create procedure sp_xuatbangdiem
    @masv varchar(20)
as
begin
    select h.tenhp, d.diemhp
    from diemhp d, dmhocphan h
    where d.mahp = h.mahp 
      and d.masv = @masv
end
SP có tác dụng xuất bảng điểm của một sinh viên cụ thể. Nó kết nối (join) hai bảng diemhp và dmhocphan thông qua mã học phần (mahp). Đồng thời, nó lọc điều kiện mã sinh viên (masv) bằng với tham số truyền vào (@masv) để chỉ hiện ra tên môn học (tenhp) và điểm số tương ứng của sinh viên đó.

Bài tập 4

Tính Điểm Trung Bình GPA: Viết SP tính điểm trung bình học tập của một sinh viên dựa trên điểm các môn và số đơn vị học trình, trả về xếp loại học lực. 

create procedure sp_tinhgpa_xeploai
    @masv varchar(20)
as
begin
    declare @tongdiem float, @tongtc int, @gpa float
    declare @xeploai varchar(20)

    select @tongdiem = sum(d.diemhp * h.sodvht), @tongtc = sum(h.sodvht)
    from diemhp d, dmhocphan h
    where d.mahp = h.mahp 
      and d.masv = @masv

    if @tongtc is null or @tongtc = 0
        set @gpa = 0
    else
        set @gpa = @tongdiem / @tongtc

    if @gpa >= 8.5 set @xeploai = 'gioi'
    else if @gpa >= 7.0 set @xeploai = 'kha'
    else if @gpa >= 5.5 set @xeploai = 'trung binh'
    else if @gpa >= 4.0 set @xeploai = 'yeu'
    else set @xeploai = 'kem'

    select @masv as masv, @gpa as gpa, @xeploai as xeploai
end
SP tính toán điểm trung bình (GPA) dựa vào trọng số tín chỉ. Khai báo các biến cục bộ (DECLARE). Tính tổng điểm = sum(điểm * số tín chỉ) và tổng tín chỉ = sum(số tín chỉ) bằng cách JOIN 2 bảng điểm và môn học. Nếu tổng tín chỉ khác 0, tính GPA = tổng điểm / tổng tín chỉ. Cuối cùng, dùng cấu trúc IF ELSE nhiều nhánh để phân loại học lực và in ra kết quả qua lệnh SELECT cuối cùng.

Bài tập 5

Thống Kê Sinh Viên Theo Lớp: Viết SP hiển thị danh sách tất cả sinh viên thuộc một lớp cụ thể và tổng số lượng sinh viên của lớp đó. 

create procedure sp_thongkesv_lop
    @malop varchar(10)
as
begin
    select * from sinhvien where malop = @malop
    
    select count(*) as tongsosinhvien from sinhvien where malop = @malop
end
SP này thực hiện hai công việc riêng biệt (2 câu SELECT liên tiếp). Đầu tiên, nó liệt kê tất cả danh sách sinh viên thuộc lớp được chỉ định bằng @malop. Sau đó, nó đếm tổng số lượng sinh viên bằng hàm COUNT(*) và đặt bí danh cho cột (tongsosinhvien).

Bài tập 6

Xóa Sinh Viên: Viết SP để xóa một sinh viên theo MaSV. Để tránh lỗi vi phạm khóa ngoại (Foreign Key), SP phải tự động xóa tất cả các điểm của sinh viên đó trong bảng DIEMHP trước, sau đó mới tiến hành xóa sinh viên trong bảng SINHVIEN. 

create procedure sp_xoasinhvien
    @masv varchar(20)
as
begin
    delete from diemhp where masv = @masv
    
    delete from sinhvien where masv = @masv
    print 'da xoa sinh vien.'
end
SP xử lý thao tác xóa sinh viên an toàn. Vì có ràng buộc khóa ngoại (sinh viên có dữ liệu ở bảng điểm), ta không thể xóa thẳng ở bảng sinhvien. SP phải thực hiện lệnh DELETE trong bảng diemhp trước để dọn sạch điểm liên quan, sau đó mới DELETE trong bảng sinhvien để hoàn tất thao tác.

Bài tập 7

Thống Kê Số Lượng Sinh Viên Theo Ngành: Viết SP nhận đầu vào là NamNhapHoc. SP sẽ trả về danh sách các Ngành học (MaNganh, TenNganh) và tổng số lượng sinh viên của từng ngành trong năm nhập học đó. 

create procedure sp_thongkesv_nganh_nam
    @namnhaphoc int
as
begin
    select n.manganh, n.tennganh, count(s.masv) as soluongsinhvien
    from dmnganh n, dmlop l, sinhvien s
    where n.manganh = l.manganh 
      and l.malop = s.malop 
      and l.namnhaphoc = @namnhaphoc
    group by n.manganh, n.tennganh
end
SP này thống kê số sinh viên mỗi ngành trong 1 năm nhập học cụ thể. Thực hiện JOIN 3 bảng: dmnganh, dmlop và sinhvien. Lọc điều kiện namnhaphoc. Dùng mệnh đề GROUP BY gom nhóm theo manganh và tennganh, đồng thời đếm số sinh viên (COUNT) trong mỗi nhóm ngành để đưa ra tổng số lượng của ngành đó.

Bài tập 8

Top N Sinh Viên Xuất Sắc Nhất Khoa: Viết SP nhận đầu vào là mã khoa (MaKhoa) và một số nguyên N. SP sẽ tính điểm trung bình của tất cả sinh viên trong Khoa đó và trả về danh sách N sinh viên có điểm trung bình cao nhất (Gồm MaSV, HoTen, TenLop, DiemTrungBinh). 

create procedure sp_topnsv_khoa
    @makhoa varchar(10),
    @n int
as
begin
    select top (@n) 
           s.masv, 
           s.hoten, 
           sum(d.diemhp * h.sodvht) / sum(h.sodvht) as diemtrungbinh
    from sinhvien s, dmlop l, dmnganh n, diemhp d, dmhocphan h
    where s.malop = l.malop 
      and l.manganh = n.manganh 
      and s.masv = d.masv 
      and d.mahp = h.mahp
      and n.makhoa = @makhoa
    group by s.masv, s.hoten
    order by diemtrungbinh desc
end
SP tìm N sinh viên xuất sắc nhất khoa. Tham số là mã khoa và số nguyên N. JOIN qua 5 bảng (sinhvien, dmlop, dmnganh, diemhp, dmhocphan) để liên kết từ sinh viên tới khoa và kết quả điểm. Tính điểm trung bình (GPA). Gom nhóm (GROUP BY) theo masv và hoten, rồi sắp xếp giảm dần bằng ORDER BY DESC. Từ khóa TOP (@n) giúp chỉ lấy ra N người đứng đầu danh sách sau khi đã sắp xếp.

Bài tập 9

Chuyển Lớp Cho Sinh Viên: Viết SP thực hiện việc chuyển lớp cho sinh viên. Đầu vào gồm MaSV và MaLop_Moi. Điều kiện: Phải kiểm tra xem MaLop_Moi có tồn tại trong bảng DMLOP hay không, và sinh viên đó có đang thực sự học ở một lớp khác không. Nếu hợp lệ thì mới tiến hành cập nhật. 

create procedure sp_chuyenlop
    @masv varchar(20),
    @malop_moi varchar(10)
as
begin
    if not exists (select * from dmlop where malop = @malop_moi)
    begin
        print 'ma lop moi khong co trong bang dmlop.'
        return
    end

    update sinhvien
    set malop = @malop_moi
    where masv = @masv

    print 'chuyen lop thanh cong.'
end
SP chuyển lớp. Dùng IF NOT EXISTS kiểm tra lớp mới truyền vào có trong hệ thống (dmlop) không. Nếu không, in ra lỗi và RETURN để kết thúc sớm (không chạy tiếp code bên dưới). Nếu hợp lệ, chạy lệnh UPDATE trên bảng sinhvien để sửa lại cột malop thành @malop_moi của sinh viên đó.

Bài tập 10

Liệt Kê Học Phần Chưa Học: Viết SP nhận vào MaSV. SP sẽ truy xuất ra danh sách các môn học (thuộc cùng MANGANH với ngành mà sinh viên đó đang theo học) nhưng sinh viên này chưa có điểm trong bảng DIEMHP. 

create procedure sp_hocphanchuahoc
    @masv varchar(20)
as
begin
    declare @manganh varchar(20)
    
    select @manganh = l.manganh 
    from sinhvien s, dmlop l
    where s.malop = l.malop and s.masv = @masv

    select mahp, tenhp
    from dmhocphan
    where manganh = @manganh
      and mahp not in (select mahp from diemhp where masv = @masv)
end
SP liệt kê môn chưa học. Bước 1: gán mã ngành sinh viên đang theo học vào biến @manganh. Bước 2: dùng câu truy vấn SELECT để lấy các môn học trong danh mục (dmhocphan) thuộc @manganh đó. Bước 3: loại trừ những môn sinh viên đã học thông qua cấu trúc NOT IN (những mã học phần đã nằm trong bảng diemhp của @masv).

Bài tập 11

Viết store procedure để thêm 1 loại. 

create procedure sp_themloai (
  @maloai int,
  @tenloai nvarchar(50),
  @mota nvarchar(max)
) as begin
insert into
  loai (maloai, tenloai, mota)
values
  (@maloai, @tenloai, @mota) end
Tạo Stored Procedure tên sp_themloai, nhận vào 3 tham số là @maloai, @tenloai và @mota. Bên trong là lệnh INSERT INTO thực hiện việc thêm một bản ghi mới vào bảng loai với các giá trị được truyền vào từ tham số.

Bài tập 12

Viết store procedure để sửa 1 loại. 

create procedure sp_sualoai (
  @maloai int,
  @tenloai nvarchar(50),
  @mota nvarchar(max)
) as begin
update loai
set
  tenloai = @tenloai,
  mota = @mota
where
  maloai = @maloai end
Tạo Stored Procedure tên sp_sualoai với 3 tham số đầu vào. Khối lệnh chứa cú pháp UPDATE để cập nhật thông tin tenloai và mota trong bảng loai. Điều kiện WHERE maloai = @maloai đảm bảo chỉ dòng dữ liệu có mã loại tương ứng bị sửa đổi.

Bài tập 13

Viết store procedure để xóa 1 loại. 

create procedure sp_xoaloai (@maloai int) as begin
delete from loai
where
  maloai = @maloai end
Tạo Stored Procedure sp_xoaloai, nhận vào 1 tham số @maloai. Lệnh DELETE FROM được sử dụng để xóa dòng dữ liệu khỏi bảng loai. Mệnh đề WHERE chỉ định rõ sẽ xóa loại hàng hóa có mã bằng với tham số truyền vào.

Bài tập 14

Viết store procedure để lấy thông tin tất cả loại. 

create procedure sp_laytatcaloai as begin

  *

Tạo Stored Procedure có tên sp_laytatcaloai, không nhận tham số đầu vào nào. Khối lệnh thực thi là câu lệnh SELECT * FROM loai để truy xuất và trả về toàn bộ dữ liệu (tất cả các cột, các dòng) có trong bảng loai.

Bài tập 15

Viết store procedure để lấy thông tin một loại cụ thể theo mã. 

create procedure sp_layloaibangma (@maloai int) as begin

Bài tập 16

Viết store procedure để lấy danh sách hàng hóa thuộc loại cụ thể. 

create procedure sp_layhanghoatheoloai (@maloai int) as begin
select
  *
from
  hanghoa
where
  maloai = @maloai end
Lệnh CREATE PROCEDURE dùng để tạo một thủ tục lưu trữ sp_layhanghoatheoloai, nhận tham số @maloai. Thủ tục này thực thi câu lệnh SELECT trên bảng hanghoa, và dùng WHERE để lọc ra tất cả hàng hóa thuộc về loại có mã truyền vào.

Bài tập 17

Viết store procedure để liệt kê DS khách hàng (HoTen, DienThoai) có đặt hàng với tổng số tiền trên X. 

create procedure sp_ds_khachhang_tongtien_tren_x (@x float) as begin
select
  kh.hoten,
  kh.dienthoai
from
  khachhang kh
  join dondathang ddh on kh.makh = ddh.makh
  join chitietdathang ct on ddh.maddh = ct.maddh
group by
  kh.makh,
  kh.hoten,
  kh.dienthoai
having
  sum(ct.soluong * ct.dongia) > @x end
Tạo Stored Procedure, nhận tham số @x. Lệnh JOIN kết nối bảng khachhang, dondathang và chitietdathang. Mệnh đề GROUP BY nhóm theo khách hàng và mệnh đề HAVING dùng hàm SUM để tính tổng tiền (số lượng * đơn giá). Cú pháp sẽ lấy ra những khách hàng có tổng tiền mua trên mức @x.

Bài tập 18

Viết store procedure để liệt kê DS hàng hóa (TenHH, Số đơn) có tổng số đơn đặt hàng trên X. 

create procedure sp_ds_hanghoa_sodon_tren_x (@x int) as begin
select
  hh.tenhh,
  count(distinct ct.maddh) as sodon
from
  hanghoa hh
  join chitietdathang ct on hh.mahh = ct.mahh
group by
  hh.mahh,
  hh.tenhh
having
  count(distinct ct.maddh) > @x end
Tạo Stored Procedure có tham số @x. Lệnh JOIN nối bảng hanghoa và chitietdathang. Dữ liệu được nhóm theo hàng hóa bằng GROUP BY. Hàm COUNT(DISTINCT ct.maddh) đếm số đơn hàng duy nhất chứa mặt hàng đó, và mệnh đề HAVING lọc ra các mặt hàng có số lượng đơn xuất hiện lớn hơn @x.

Bài tập 19

Viết store procedure để liệt kê DS hàng hóa (TenHH, Tổng SL) có tổng số lượng đặt hàng trên X. 

create procedure sp_ds_hanghoa_tongsl_tren_x (@x int) as begin
select
  hh.tenhh,
  sum(ct.soluong) as tongsl
from
  hanghoa hh
  join chitietdathang ct on hh.mahh = ct.mahh
group by
  hh.mahh,
  hh.tenhh
having
  sum(ct.soluong) > @x end
Tạo Stored Procedure nhận tham số @x. Thực hiện JOIN giữa hanghoa và chitietdathang, sau đó GROUP BY theo mặt hàng. Hàm tập hợp SUM(ct.soluong) tính tổng số lượng của hàng hóa được đặt, mệnh đề HAVING giữ lại những hàng hóa có tổng số lượng bán ra lớn hơn @x.

Bài tập 20

Viết store procedure để liệt kê DS hàng hóa (TenHH, Tổng số tiền) có tổng số tiền đặt hàng trên X. 

create procedure sp_ds_hanghoa_tongtien_tren_x (@x float) as begin
select
  hh.tenhh,
  sum(ct.soluong * ct.dongia) as tongtien
from
  hanghoa hh
  join chitietdathang ct on hh.mahh = ct.mahh
group by
  hh.mahh,
  hh.tenhh
having
  sum(ct.soluong * ct.dongia) > @x end
Tạo Stored Procedure với tham số @x. Lệnh sử dụng JOIN để kết hợp bảng hanghoa và chitietdathang, đồng thời dùng GROUP BY để gom nhóm theo hàng hóa. Mệnh đề HAVING so sánh tổng tiền thu được từ hàng hóa (tính bằng hàm SUM) với tham số @x, nhằm lọc ra các mặt hàng mang lại doanh thu lớn hơn giá trị này.

🛒 CSDL: Quản Lý Bán Hàng (QUANLYBANHANG)

Bài tập 1

Hiển thị danh các khách hàng đã mua hàng trong ngày chỉ định (ngày là tham số truyền vào). 

CREATE PROCEDURE p_KhachHangTheoNgay @NgayLap date AS BEGIN
SELECT
  KH.*
FROM
  KHACHHANG KH
  INNER JOIN HOADON HD ON KH.MaKH = HD.MaKH
WHERE
  HD.NgayLapHD = @NgayLap END
Tạo thủ tục nhận tham số ngày và thực hiện câu lệnh SELECT với tham số đó.

Bài tập 2

Hiển thị danh sách 5 khách hàng có tổng trị giá các đơn hàng lớn nhất. 

CREATE PROCEDURE p_Top5KhachHang AS BEGIN
SELECT
  TOP 5 KH.MaKH,
  KH.TenKH,
  SUM(C.SoLuong * H.DonGia) AS TongTriGia
FROM
  KHACHHANG KH
  INNER JOIN HOADON HD ON KH.MaKH = HD.MaKH
  INNER JOIN CHITIETHD C ON HD.MaHD = C.MaHD
  INNER JOIN HANGHOA H ON C.MaH = H.MaH
GROUP BY
  KH.MaKH,
  KH.TenKH
ORDER BY
  SUM(C.SoLuong * H.DonGia) DESC END
Sử dụng TOP 5 và ORDER BY DESC để lấy 5 người chi tiêu nhiều nhất.

Bài tập 3

Hiển thị danh sách 10 mặt hàng có số lượng bán lớn nhất. 

CREATE PROCEDURE p_Top10MatHang AS BEGIN
SELECT
  TOP 10 H.MaH,
  H.TenH,
  SUM(C.SoLuong) AS TongSoLuong
FROM
  HANGHOA H
  INNER JOIN CHITIETHD C ON H.MaH = C.MaH
GROUP BY
  H.MaH,
  H.TenH
ORDER BY
  SUM(C.SoLuong) DESC END
Tính tổng số lượng của mỗi mặt hàng, sắp xếp giảm dần và lấy TOP 10.

Bài tập 4

Cập nhật cột Khuyến mãi như sau: Khuyến mãi 5% thành tiền nếu SoLuong >100, 10% thành tiền nếu SoLuong >500. 

UPDATE CHITIETHD
SET
  KhuyenMai = CASE
    WHEN SoLuong > 500 THEN (SoLuong * DonGia) * 0.10
    WHEN SoLuong > 100 THEN (SoLuong * DonGia) * 0.05
    ELSE 0
  END
FROM
  CHITIETHD C
  INNER JOIN HANGHOA H ON C.MaH = H.MaH
Sử dụng lệnh UPDATE kết hợp cấu trúc CASE WHEN.

Bài tập 5

Tính trị giá cho mỗi hoá đơn. 

CREATE PROCEDURE p_TriGiaHoaDon AS BEGIN
SELECT
  C.MaHD,
  SUM(C.SoLuong * H.DonGia) AS TriGia
FROM
  CHITIETHD C
  INNER JOIN HANGHOA H ON C.MaH = H.MaH
GROUP BY
  C.MaHD END
Thủ tục đơn giản sử dụng GROUP BY trên chi tiết hóa đơn.

Bài tập 6

Cập nhật cho cột Loại khách hàng: là VIP nếu tổng thành tiền trong năm lớn hơn hoặc bằng 20 triệu. 

UPDATE KHACHHANG
SET
  LoaiKH = 'VIP'
WHERE
  MaKH IN (
    SELECT
      HD.MaKH
    FROM
      HOADON HD
      INNER JOIN CHITIETHD C ON HD.MaHD = C.MaHD
      INNER JOIN HANGHOA H ON C.MaH = H.MaH
    WHERE
      YEAR(HD.NgayLapHD) = YEAR(GETDATE())
    GROUP BY
      HD.MaKH
    HAVING
      SUM(C.SoLuong * H.DonGia) >= 20000000
  )
Sử dụng UPDATE với điều kiện lọc IN chứa câu truy vấn tính tổng thành tiền.
