Ghi Chú AI Profile

Viết ghi chú ở đây để AI tập trung vào một chủ đề hoặc phong cách cụ thể. Nội dung này sẽ được gửi kèm mỗi khi bạn chat.

Hướng Dẫn Sử Dụng AI Chat

🚀 Bắt đầu nhanh
  1. Bấm ⚙️ Cài đặt (bánh răng) → nhập Groq API Key để dùng các model Llama, hoặc Gemini API Key để dùng Gemini.
  2. Chọn Model AI phù hợp từ danh sách thả xuống ở thanh trên.
  3. Gõ câu hỏi SQL, đính kèm file/ảnh nếu cần, rồi bấm Gửi.
🧠 AI đã biết gì?

AI đã được nạp sẵn toàn bộ kiến thức về CSDL QLSV_NEW, bao gồm:

  • Cấu trúc 6 bảng (DMKHOA, DMNGANH, DMLOP, SINHVIEN, DMHOCPHAN, DIEMHP)
  • Kiểu dữ liệu, Khóa chính, Khóa ngoại
  • Mối quan hệ giữa các bảng
💡 Mẹo sử dụng hiệu quả
  • Hỏi cấu trúc: "Bảng SINHVIEN có những cột nào?"
  • Nhờ giải bài: "Giúp tôi giải bài tập 3 phần Truy Vấn Cơ Bản"
  • Hỏi lý thuyết: "INNER JOIN khác LEFT JOIN chỗ nào?"
  • Debug lỗi: Dán đoạn SQL bị lỗi vào và hỏi "Sửa giúp tôi đoạn này"
  • Đính kèm file: Bấm 📎 để gửi file .sql hoặc hình ảnh cho AI phân tích
👤 AI Profile

Bấm nút 👤 Profile để viết ghi chú tuỳ chỉnh. Ví dụ: "Tôi đang ôn thi chương Trigger, hãy tập trung giải thích phần này". AI sẽ nhớ và ưu tiên chủ đề bạn yêu cầu!

⚡ Các Model AI đang dùng
  • Llama 3.3 70B: model Groq mạnh nhất trong danh sách, phù hợp giải thích bài khó, viết truy vấn nhiều bước, phân tích JOIN/GROUP BY/Procedure/Trigger.
  • Llama 3.1 8B Instant: model Groq nhẹ và rất nhanh, phù hợp hỏi đáp ngắn, sửa lỗi SQL đơn giản, nhắc cú pháp nhanh.
  • Gemini 3.5 Flash: model Google nhanh, hỗ trợ ảnh, phù hợp giải thích tổng quát và xử lý câu hỏi dài.
