2. Stored Procedures (Thủ Tục Lưu Trữ)

Stored Procedure giống như một cái "hàm" trong lập trình (ví dụ hàm C++). Bạn gom một đống câu lệnh SQL (INSERT, UPDATE, DELETE, SELECT) vào thành một cục có tên đàng hoàng. Sau này cần dùng chỉ cần gọi tên nó ra (bằng lệnh EXEC) và truyền tham số vào là xong.

Cú pháp CỰC KỲ QUAN TRỌNG: Không dùng ngoặc đơn ()

Khác với C/C++ hay Java, trong SQL Server khi khai báo tham số của Procedure, bạn KHÔNG được dùng dấu ngoặc đơn () bọc các biến lại. Khai báo biến phải bắt đầu bằng chữ @.

2.1. Cú pháp cơ bản

CREATE PROCEDURE ten_thu_tuc
    @tham_so_1 KIEU_DU_LIEU,
    @tham_so_2 KIEU_DU_LIEU
AS
BEGIN
    -- Các câu lệnh SQL nằm ở đây
END
GO

2.2. Ví dụ 1: Procedure kiểm tra điều kiện rồi mới INSERT

Bài toán (Từ bài TH Buổi 3): Thêm một sinh viên mới, nhưng trước khi thêm phải kiểm tra xem mã lớp có tồn tại trong bảng DMLOP hay chưa. Nếu chưa có thì báo lỗi.

CREATE PROCEDURE sp_ThemSinhVien
    @MaSV VARCHAR(20),
    @HoTen NVARCHAR(50),
    @MaLop VARCHAR(10)
AS
BEGIN
    -- Kiểm tra lớp có tồn tại không
    IF EXISTS (SELECT * FROM DMLOP WHERE MALOP = @MaLop)
    BEGIN
        -- Nếu tồn tại (IF đúng) -> Chạy INSERT
        INSERT INTO SINHVIEN (MASV, HOTEN, MALOP)
        VALUES (@MaSV, @HoTen, @MaLop)
        PRINT N'Thêm sinh viên thành công!'
    END
    ELSE
    BEGIN
        -- Nếu không tồn tại -> Báo lỗi
        PRINT N'Mã lớp không tồn tại, không thể thêm.'
    END
END
GO

-- CÁCH SỬ DỤNG (Gọi bằng chữ EXEC)
EXEC sp_ThemSinhVien '099', N'Nguyễn Văn A', 'CT11'
GO

2.3. Ví dụ 2: Khai báo biến trung gian để tính toán

Bạn có thể tự tạo ra một cái hộp chứa (biến trung gian) trong thân Procedure bằng lệnh DECLARE để tính toán trước khi SELECT in ra kết quả.

CREATE PROCEDURE sp_ThongKeSV_Lop
    @MaLop VARCHAR(10)
AS
BEGIN
    -- Khai báo biến trung gian kiểu số nguyên (INT)
    DECLARE @TongSoLuong INT;
    
    -- Đếm và bỏ kết quả vào cái hộp @TongSoLuong
    SELECT @TongSoLuong = COUNT(*) FROM SINHVIEN WHERE MALOP = @MaLop;
    
    -- In ra kết quả
    PRINT N'Lớp ' + @MaLop + N' có tổng cộng: ' + CAST(@TongSoLuong AS VARCHAR) + N' sinh viên.';
    
    -- In ra danh sách chi tiết
    SELECT * FROM SINHVIEN WHERE MALOP = @MaLop;
END
GO

Hoàn thành bài học để tăng tiến độ khóa học SQL của bạn!
